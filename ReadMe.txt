Программа - лабораторная работа по дисциплине ООКП. Представляет собой некий набор функционала для доступа к данным БД. Реализует чтение, запись, изменение, сортировку данных. Отлавливает некоторые возможные ошибки в процессе выполнения.

Работает программа непосредственно с программой MySQLWorkbench, в который был создан локальный сервер для доступа к БД.
